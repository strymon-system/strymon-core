pub struct ProcessService { /* fields omitted */ }

Strymon job process spawning and supervision service.

This structure is responsible for keeping track of the currently running child processes, their output and handles potential termination requests.

All supervised children are terminated when the tokio event loop is dropped, though this might change in the future.
